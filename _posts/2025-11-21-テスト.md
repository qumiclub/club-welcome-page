## 1. リスト表現

### 箇条書きリスト (Unordered List)
- リストアイテム 1
- リストアイテム 2
  - ネストされたアイテム 2-1
  - ネストされたアイテム 2-2
    - さらにネスト 2-2-1
- リストアイテム 3

### 番号付きリスト (Ordered List)
1. 手順その1
2. 手順その2
3. 手順その3
   1. サブ手順 1
   2. サブ手順 2

### タスクリスト (Task List)
- [x] 完了したタスク
- [ ] 未完了のタスク
- [ ] 進行中のタスク

### コードブロック (Code Block)
以下はPythonコードのシンタックスハイライトのテストです。

import numpy as np

def sigmoid(x):
    """シグモイド関数を計算する"""
    return 1 / (1 + np.exp(-x))

print(sigmoid(0)) # 出力: 0.5
