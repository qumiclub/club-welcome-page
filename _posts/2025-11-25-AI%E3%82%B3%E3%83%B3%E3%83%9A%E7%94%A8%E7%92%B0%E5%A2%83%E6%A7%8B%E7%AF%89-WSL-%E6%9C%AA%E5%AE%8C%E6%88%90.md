Windows 上で AI コンペ向けの開発環境を構築する手順をまとめました。Python / PyTorch / VSCode を中心に、WSL2 での作業を前提としています。

---

## 1. WSL2 セットアップ

### 1.1 WSL2 インストール

1. **Windows機能の有効化**
   管理者権限で PowerShell を開き、以下を実行：

   ```powershell
   wsl --install
   ```
2. **ディストリビューションの選択**
   Ubuntu 22.04 LTS を例に：

   ```powershell
   wsl --install -d Ubuntu-22.04
   ```
3. **WSL2 バージョン確認**

   ```powershell
   wsl -l -v
   ```

---

### 1.2 必要ツールのインストール 例

WSL 内で以下を実行：

```bash
sudo apt update
sudo apt install -y curl git build-essential unzip
```

* **curl**: ファイルダウンロード用
* **git**: ソースコード管理
* **unzip**: 圧縮展開用
* **build-essential**: Python パッケージビルド用

## 2. VSCode セットアップ

### 2.1 VSCode インストール

公式サイトから Windows 版をインストール：[VSCode ダウンロード](https://code.visualstudio.com/)

---

### 2.2 Remote - WSL 拡張

1. VSCode の拡張機能から **Remote - WSL** をインストール
2. WSL 内ディストリビューションを直接開くことが可能

---

### 2.3 ターミナル設定

* デフォルトターミナルを WSL に設定：

  * 設定 → Terminal → Integrated → Default Profile → `WSL`
* 推奨シェル: `zsh` + `oh-my-zsh`（作業効率向上）

```bash
sudo apt install -y zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
chsh -s $(which zsh)
```

---

### 2.4 推奨拡張機能

* Python
* Pylance
* Jupyter
* Git Graph
* Docker（必要に応じて）

## 3. プロジェクト初期化

### 3.1 PyTorch インストール

GPU 対応の場合は CUDA バージョンに合わせて：

```bash
$ # With a command-line argument.
$ uv pip install torch --torch-backend=auto

```


確認：

```python
import torch
print(torch.cuda.is_available())
```

---

### 3.2 データセット配置と I/O

1. ディレクトリ作成：

```bash
mkdir -p ~/ai_competition/data
cd ~/ai_competition/data
```

2. WSL 内に直接ダウンロード：

```bash
curl -O https://example.com/dataset/train.csv
curl -O https://example.com/dataset/test.csv
```

* 大容量の場合は `wget` が便利：

```bash
wget https://example.com/dataset/large_dataset.zip
unzip large_dataset.zip
```

3. Python からの読み込み：

```python
import pandas as pd

train_df = pd.read_csv('~/ai_competition/data/train.csv')
test_df  = pd.read_csv('~/ai_competition/data/test.csv')
```

* WSL 内に置くことで I/O が高速化されます。
* プロジェクト内の `data/` フォルダを整理しておくと便利。
